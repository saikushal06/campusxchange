# 🚀 CampusXchange

A modern realtime campus marketplace platform built for students to buy, sell, exchange, and communicate securely within their campus ecosystem.

---

## 🌐 Live Demo

🔗 https://campusxchange.saikushal467.workers.dev

---

# ✨ Features

## 🔐 Authentication
- Google Sign-In Authentication
- Secure Firebase Auth integration
- Protected user routes

## 🛒 Marketplace System
- Create listings
- Edit/Delete listings
- Browse products
- Wishlist system
- Seller profiles

## 💬 Realtime Messaging
- Realtime chat system
- Typing indicators
- Online/offline presence
- Last message updates

## ⭐ Trust & Safety
- Seller reviews & ratings
- Dynamic seller reputation
- Reporting system
- Admin moderation dashboard
- Secure Firestore rules

## 🔍 Advanced UX
- Smart search suggestions
- Premium analytics dashboard
- Skeleton loading states
- Responsive mobile-first UI
- Dark/Light mode support

## 📱 PWA Support
- Installable web app
- Mobile-friendly experience
- Native-app feel
